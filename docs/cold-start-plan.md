# 拼豆图纸生成器 · 冷启动方案

> 基于：小红书 80 亿+ 浏览 / Etsy 7 倍溢价 / 纯客户端产品架构
> 目标：90 天起盘，小红书种草 + Etsy 变现双轨并行

---

## 一、市场洞察

### 1.1 小红书：拼豆正处于流量爆发期

| 指标 | 数据 |
|------|------|
| 话题总浏览 | 80 亿+ |
| 近 30 天新增互动 | 4000 万+ |
| 赛道对比 | 超过美妆、穿搭、家居 |

**结论：拼豆在小红书是 2025-2026 最热赛道，没有之一。**

### 1.2 Etsy：成熟市场，高溢价

| 特征 | 数据 |
|------|------|
| 市场历史 | 20+ 年（Perler 品牌 1981 年成立） |
| 用户画像 | 老手多，自主创作，不依赖预设图案 |
| 溢价空间 | 单色豆 2000 粒 $5（约为淘宝 7 倍） |
| 热门品类 | 定制 DIY Kit、数字图纸下载、节日套装 |

**结论：Etsy 用户愿意为创意和便利付费，图纸下载（Digital Download）是最低成本切入方式。**

### 1.3 我们的差异化

| 竞品 | 我们 |
|------|------|
| 手动选色/凭感觉 | AI 自动像素化 + 颜色映射 |
| 套装预设图案 | 任意图片转图纸 |
| 无配料计算 | 自动 BOM（含重量/冗余/配件） |
| 无品牌感 | 专业 PDF 多页文档（封面/配料表/图纸/标签） |

## 二、产品包装策略

### 2.1 小红书侧：图纸分享

**产品形态：PNG 图纸 + 色号统计图**

- 利用现有 `imageDownloader.ts` 导出
- 图片底部品牌水印 `@LDB`（已有）
- 优化：在标题栏增加「AI 生成」标识，强化科技感

**内容矩阵（5 个方向）：**

| 内容类型 | 频率 | 示例标题 | 目标 |
|----------|------|----------|------|
| AI 演示 | 3/周 | "上传图片 3 秒出图纸？拼豆 AI 生成器实测" | 引流 |
| 成品展示 | 2/周 | "AI 设计的 Labubu 拼豆，朋友以为是买的" | 种草 |
| 教程 | 2/周 | "拼豆怎么熨烫不粘？99% 的人都做错了" | 收藏 |
| 热点 | 随势 | "原神/Chiikawa 最新图纸来了" | 爆量 |
| 幕后 | 1/周 | "用 AI 给女儿做了个艾莎公主拼豆" | 信任 |

### 2.2 Etsy 侧：数字图纸商店

**产品形态：PDF 多页文档**

- 利用现有 `pdfExporter.ts` 导出
- 在 PDF 封面嵌入二维码（已有，`/website_qrcode.png`）
- 二维码指向：AI 图纸生成器网站

**商品类型：**

| 商品 | 定价 | 说明 |
|------|------|------|
| 单张图纸 | $3-5 | PDF 含封面/配料表/图纸/标签 |
| 系列图纸包（5 张） | $12-18 | 主题系列（节日/角色/动物） |
| 定制图纸 | $15-25 | 买家发图，AI 生成后出 PDF |

**上架节奏：**

- **Day 1-30**：上架 20 款免费图纸（引流款，建立店铺权重）
- **Day 31-60**：上架 30 款付费图纸 + 5 个系列包
- **Day 61-90**：推出定制服务（利用 AI 降低交付成本）

## 八、风险与对策

| 风险 | 概率 | 对策 |
|------|------|------|
| 小红书引流被判定广告 | 高 | 内容以教程/展示为主，产品推荐穿插 |
| Etsy 同质化竞争 | 中 | AI 差异化 + 低价数字品引流 |
| 配料包供应链不稳定 | 中 | 先手动分装，验证需求后上自动化 |
| 用户不买账（不转化） | 中 | 免费图纸养店，数据说话 |
| IP 侵权风险 | 中 | 定制图纸免责声明，避免主动销售角色图纸 |
